Run a JPEG simulation from the command line

As an alternative to using the JPG Simulation tool UI, run a JPEG simulation via the command line. To expedite JPEG simulation from multiple images, create and run a batch file with multiple commands.

Run a batch file with multiple JPG input files

Use these steps to run a JPEG command line simulation with files from an existing tuned project and JPG files as input.

  1. Open the command prompt.
  2. Navigate to the Chromatix directory.
  3. To enable the command line JPG simulation, type something like the following: 
    -JPGSimAll 
-awbdll C:\JPGSim_CommandLine\input\AWBSim\com.qti.stats.awb.dll 
-bin C:\JPGSim_CommandLine\input\AWBSim\com.qti.tuned.liteon_imx318.bin 
-jpg C:\JPGSim_CommandLine\input\2DL_Y=1.jpg 
-xsd C:\JPGSim_CommandLine\input\XSD 
-xml C:\JPGSim_CommandLine\input\XML 
-inputMeta C:\JPGSim_CommandLine\input\meta_config_template.xml 
-outputFolder C:\JPGSim_CommandLine\output
    Table : JPGSimAll command structure
    Argument Description
    -JPGSimAll Enable JPG simulation mode to use in batch file
    -awbdll <filename> Path to the awb dll file in the software build
    -bin <filename> Path to the tuned project binary file for the target device
    -jpg <filename> Path to the JPEG image that has the 3A debug data
    -xsd <folder name> Path to the XSD folder for the tuned project
    -xml <folder name> Path to the XML folder for the tuned project
    -inputMeta <filename> Path to the input metadata file to use as meta template file. Create this the first time by running the JPGCfg command (see below).
    -outputFolder <folder name> Path to the output folder for the updated metadata file
  4. Add similar set of commands for each JPEG file that you want to use for JPG simulation. Put the sets into a batch file.
  5. Run the batch file.

Other JPEG commands to generate related meta/config files

Table : JPGCfg command structure
Argument Description
-JPGCfg Generate JPG configuration and meta info files for jpg simulation
-jpg <filename> Path to the JPEG image that contains both 3A debug info and metadata
-OutputCfg <filename> Path to the JPEG configuration txt file, used for JPG simulation
-inputMeta <filename> Path to the input metadata xml file to use as a template for generating output
-outputMeta <filename> Path to the output metadata xml file for JPG simulation with updated cct and rgb gains
Table : JPGCfg command structure
Argument Description
-ProjCfg Generate project configuration file for JPG simulation
-xsd <folder name> Path to the XSD folder for the tuned project
-xml <folder name> Path to the XML folder for the tuned project
-inputMeta <filename> Path to the input metadata xml file, could be output of AWB simulation
-outputCfg <filename> Path to the output project configuration file for JPG simulation
Table : JPGSim command structure
Argument Description
-JPGSim Generate JPG simulation
-jpg <filename> Path to the JPEG image that contains both 3A debug info and metadata
-jpgCfg <filename> Path to the JPEG configuration txt file
-projCfg <filename> Path to the project configuration txt file
-sourceMeta <filename> Path to source metadata file that contains r,g,b gains from jpg file
-targetMeta <filename> Path to the source metadata file for new r,g,b gains
-outputFolder <folder name> Path to the JPG simulation output folder
