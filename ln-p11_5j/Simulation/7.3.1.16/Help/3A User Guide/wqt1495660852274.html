Monitor algo config parameters

There are four monitor algo configs by default. One for each of the default monitor algorithms (Topo CAF, Topo PDAF, Topo TOF, and Topo DCIAF). Because each algorithm may rely on different stats, there are different sensitivity parameters for each config. The following tables list the parameters for each monitor algo config.

NOTE: Setting a sensitivity parameter to 0 disables the parameters in the corresponding value monitor bank. For example, setting SADR to 0 disables the parameters in SADR VM set type. The output decision in such cases is 0.
