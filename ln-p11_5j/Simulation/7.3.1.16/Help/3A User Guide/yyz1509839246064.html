Fine-tune face-assist scene analyzer

Use the procedures in this section to fine-tune the AWB face-assist scene analyzer parameters. Review the FA SA concepts to determine which submodule to tune.

Tune basic face-assist AWB parameters

  1. Open the project and click the AWB tab.
  2. In the List View section, select Face Assist AWB.
  3. In the Modules section, select Basic Tuning.

  4. In the Basic Tuning section, adjust the following parameters as needed:.
    • ROI X scale
    • ROI Y scale
    • Face select threshold
    • Face detection count threshold
    • No face detection count threshold
    • Convergence speed for face AWB
  5. Click File > Save To Project to save the changes.

Tune or add dark pixel threshold triggers

Adjust the dark pixel threshold triggers to exclude more or fewer dark stats in the BG decision for a face-dominated scene. If stats brightness is lower than this threshold, the stats are not considered for face stats.

  1. Open the project and click the AWB tab.
  2. On the List View tab, select Face Assist AWB.
  3. In the Modules box, select Dark/Saturation Pixel Threshold.
  4. Find the Dark Pixel Threshold section.

  5. To edit the lux index trigger range, in the first column, double-click the entry in brackets. Update the lux index start and end values and click outside the box to save the change.
  6. To edit the red/green/blue dark pixel threshold data, in the second column, double-click the entry in brackets. Update the red/green/blue data values for this trigger and click outside the box to save the change.
  7. To add more trigger levels, use the Add above or Add below buttons to add a new blank trigger row. Repeat the previous three steps to set the trigger range and associated core data.
  8. Click File > Save To Project to save the changes.

Tune or add light skin WB target triggers

If the AWB face assist computation of light skin WB target points is not adequate, use this procedure to fine tune the target. Refer to the example in FA SA estimation of skin CCT. For example, if the overall light skin is reddish, reduce the WB red target.

  1. Open the project and click the AWB tab.
  2. On the List View tab, select Face Assist AWB.
  3. In the Modules box, select Light Skin WB Target.
  4. In the Trigger Tuning section, notice that the first trigger is Lux Idx and the second trigger is CCT.
  5. Click the arrow next to the first trigger range to open the full set of values, as shown in the image.

  6. To edit the lux index range, in the first column, double-click the entry in brackets. Update the lux index start and end values and click outside the box to save the change.
  7. To edit the CCT range, in the second column, double-click the entry in brackets. Update the CCT start and end values and click outside the box to save the change.
  8. To edit the red/blue WB core data, in the third column, double-click the entry in brackets. Update the red/blue data values for this trigger and click outside the box to save the change.
    Note: Fine tune the WB target for preference. For example, if the overall light skin color is reddish, try reducing the WB r gain target.
  9. To add more triggers, use the Add above or Add below buttons to add a new blank trigger row. Repeat the previous three steps to set the trigger range and associated core data.
  10. Click File > Save To Project to save the changes.
