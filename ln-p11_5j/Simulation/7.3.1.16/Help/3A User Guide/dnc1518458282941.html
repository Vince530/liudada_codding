Fine tune AWB dual camera sync weight FCM

Prerequisite: Complete the initial AWB tuning process for each camera in its own Chromatix project. In the initial AWB tuning, measure the regular reference points of both cameras under the same light condition.

Fine tune weight FCM

It is recommended to use the default value of 0 (GCM focused) except when the gcmNumRatio is low. However, it is a good practice to increase the weight of the FCM when the FCM result is better in a certain Lux Index and CCT environment due to the very similar spectral response of the master camera and the slave camera in the condition.

  1. Open the project and click the AWB tab.
  2. In the Parameter Editor, navigate to 3A > AWB > Dual Cam Sync (V1) > Weight FCM to view the existing outlier distance configuration. Determine what additional trigger structures or value changes are needed.
  3. To add a new first level (LuxIdx) trigger range, do the following:
    1. In the left pane, navigate to 3A > AWB > Dual Cam Sync (V1) > Weight FCM and select Trigger.
    2. In the right pane, click the plus sign (+) next to the row under which you want to add a blank trigger structure (for example, click + next to Trigger [0] to create Trigger [1]). Add as many first level trigger structures as required.
    3. In each new LuxIdx trigger structure row, click the dots in the Value column to set the LuxIdx Start/End values. Repeat for each first level structure you added.

      For each first level (LuxIdx) trigger, the system sets up one second level (CCT) trigger structure.

  4. Configure new second level (CCT) triggers as follows:
    1. In the left pane, navigate to 3A > AWB > Dual Cam Sync (V1) > Weight FCM > Trigger[x] and select Trigger.
    2. To add an additional second level (CCT) trigger for the same LuxIdx range, in the right pane, click the plus sign (+) next to the row under which you want to add a blank trigger structure (for example, click + next to Trigger [0] to create Trigger [1]). Add as many second level trigger structures as required.
    3. In the right pane, click the dots in the Value column of the Trigger Pt row to set the CCT Start/End values for each second level structure you added.

      For each second level (CCT) trigger, the system creates one outlier distance data field.

  5. Configure new third level (gcmNumRatio) triggers as follows:
    1. In the left pane, navigate to 3A > AWB > Dual Cam Sync (V1) > Weight FCM > Trigger[x] > Trigger[x] and select Trigger.
    2. To add an additional third level (gcmNumRatio) trigger for the same CCT range, in the right pane, click the plus sign (+) next to the row under which you want to add a blank trigger structure (for example, click + next to Trigger [0] to create Trigger [1]). Add as many third level trigger structures as required.
    3. In the right pane, click the dots in the Value column of the Trigger Pt row to set the gcmNumRatio Start/End values for each third level structure you added.

      For each third level (gcmNumRatio) trigger, the system creates one weight FCM data field.

  6. Configure the weight FCM value associated with the third level trigger as follows:
    1. In the left pane, navigate to 3A > AWB > Dual Cam Sync (V1) > Weight FCM > Trigger[x] > Trigger[x] > Trigger[x].
    2. In the right pane, click the dots in the Value column of the Trigger row and enter the weight FCM value.
  7. To update an existing trigger configuration, navigate to 3A > AWB > Dual Cam Sync (V1) > Weight FCM and do the following:
    • To adjust the Lux Index trigger range, navigate to Trigger[x] > Trigger Pt (LuxID) and make the necessary changes.
    • To adjust the CCT trigger range, navigate to Trigger[x] > Trigger[x] > Trigger Pt (CCT) and make the necessary changes.
    • To adjust the gcmNumRatio trigger range, navigate to Trigger[x] > Trigger[x] > Trigger[x] > Trigger Pt (gcmNumRatio) and make the necessary changes.
    • To adjust the weight FCM value, navigate to Trigger[x] > Trigger[x] > Trigger[x] > Trigger (Data) and make the necessary changes.
  8. To delete an existing trigger configuration, navigate to 3A > AWB > Dual Cam Sync (V1) > Weight FCM and do the following:
    • Navigate to Trigger[x] > Trigger[x]and click the minus sign (-) to delete an existing CCT trigger.
    • Navigate to Trigger[x] and click the minus sign (-) to delete an existing Lux Index trigger.
  9. Click File > Save to project.
